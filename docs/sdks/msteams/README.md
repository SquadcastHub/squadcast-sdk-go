# Msteams
(*Msteams*)

## Overview

### Available Operations

* [GetConfig](#getconfig) - Get MSTeams Config

## GetConfig

Returns MSTeams config of the organization.
Requires `access_token` as a `Bearer {{token}}` in the `Authorization` header with `read` scope.
